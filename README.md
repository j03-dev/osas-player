# ossas-MusicPlayer

![screenshot](image.png)

## how to install

```bash
git clone github.com/j03-dev/ossas-MusicPlayer
cd ossas-MusicPlayer
python3 -m pip install -r requirement.txt
```

## config

_open_ **.config** and change<br>

```
path = your music directory
```

## for run ossas

```bash
python3 ossas.py
```
